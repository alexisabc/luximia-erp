# Atoms (Átomos)

Componentes básicos e indivisibles del sistema de diseño.

## Características
- **Indivisibles**: No se pueden descomponer en componentes más pequeños
- **Reutilizables**: Usados en múltiples contextos
- **Sin lógica de negocio**: Solo presentación
- **Altamente configurables**: Mediante props

## Ejemplos
- Button
- Input
- Label
- Icon
- Badge
- Avatar
- Spinner

## Convenciones
- Nombres en PascalCase
- Props bien documentadas
- Variantes mediante props (no múltiples componentes)
- Mobile First en todos los estilos
