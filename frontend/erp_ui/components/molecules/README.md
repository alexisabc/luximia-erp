# Molecules (Moléculas)

Combinaciones simples de átomos que forman componentes funcionales.

## Características
- **Composición de átomos**: Combinan 2-5 átomos
- **Funcionalidad específica**: Tienen un propósito claro
- **Reutilizables**: En diferentes contextos
- **Lógica simple**: Manejo de estado local básico

## Ejemplos
- SearchBar (Input + Icon)
- FormField (Label + Input + ErrorMessage)
- ActionButton (Button + Icon)
- StatusBadge (Badge + Icon)
- DatePicker (Input + Calendar Icon + Dropdown)

## Convenciones
- Nombres descriptivos de su función
- Composición sobre configuración
- Props para personalización
- Mobile First
