# Templates (Plantillas)

Layouts que definen la estructura de las páginas.

## Características
- **Estructura de página**: Definen el layout general
- **Sin datos reales**: Usan placeholders
- **Composición de organismos**: Ensamblan organismos
- **Reutilizables**: Para múltiples páginas similares

## Ejemplos
- DashboardTemplate (Sidebar + Header + MainContent)
- FormTemplate (Header + FormSection + Actions)
- ListTemplate (Header + Filters + DataTable)
- DetailTemplate (Header + InfoCard + RelatedData)

## Convenciones
- Sufijo "Template" en el nombre
- Props para slots de contenido
- Responsive layouts
- Mobile First con adaptaciones
