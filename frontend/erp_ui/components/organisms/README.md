# Organisms (Organismos)

Componentes complejos que forman secciones completas de la interfaz.

## Características
- **Composición compleja**: Combinan átomos y moléculas
- **Secciones completas**: Header, Sidebar, DataTable, etc.
- **Lógica de negocio**: Pueden tener estado complejo
- **Contexto específico**: Relacionados con el dominio

## Ejemplos
- Header (Logo + Navigation + UserMenu)
- Sidebar (Navigation + MenuItems + UserProfile)
- DataTable (SearchBar + Table + Pagination)
- FormSection (Multiple FormFields + Actions)
- DashboardCard (Card + Chart + Actions)

## Convenciones
- Nombres que reflejan su rol en la UI
- Pueden consumir contextos
- Manejo de estado complejo
- Responsive y Mobile First
